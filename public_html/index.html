Keep your site. Add a modern publishing core.

Hybrid-Static-CMS is built for teams with existing HTML or PHP sites that need structured content without surrendering the whole document root to a monolithic CMS.

Open control panel View latest fragment

Static fragments

Posts are stored in PostgreSQL, rendered into static HTML fragments, RSS feeds, and sitemaps.

REST API

A full JSON API lets you fetch posts, pages, media, and forms from any client or framework.

Media library

Upload and manage images, videos, audio, and documents with automatic embed snippets.

Role-based access

Fine-grained permissions with owner, admin, editor, and viewer roles for team collaboration.

File snapshots

Create backups of your public_html files before making changes, with one-click restore.

Coexistence model

Existing pages use HTML includes, script embedding, or fetch APIs while keeping the original site structure intact.
