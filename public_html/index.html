Hybrid-Static-CMS is built for teams with existing HTML or PHP sites that need structured content without surrendering the whole document root to a monolithic CMS.

How it works

Posts are stored in PostgreSQL, rendered into static fragments, and exposed via API.

Existing pages can use HTML includes, script embedding, or fetch APIs while keeping the original site structure intact.
